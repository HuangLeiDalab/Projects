======================================================================

These files contain measured material parameters obtained from cloth 
samples.  They are stored in a format suitable for use with the cloth 
simulator ARCSim. (http://graphics.berkeley.edu/resources/ARCSim)

Any scholarly publication using these data shall cite:

  Huamin Wang, Ravi Ramamoorthi, and James F. O'Brien. "Data-Driven 
  Elastic Models for Cloth: Modeling and Measurement". ACM 
  Transactions on Graphics, 30(4):71:1â€“11, July 2011. Proceedings of 
  ACM SIGGRAPH 2011, Vancouver, BC Canada.
